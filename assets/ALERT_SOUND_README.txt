# Alert Sound Placeholder

This directory should contain an alert.mp3 file.

You can:
1. Use the included placeholder
2. Replace with your own alert sound (short, attention-grabbing sound recommended)

The audio file should be:
- Format: MP3
- Duration: 1-3 seconds
- Volume: Clear and audible
- Name: alert.mp3

You can find free alert sounds at:
- https://freesound.org
- https://mixkit.co/free-sound-effects/alert/

Or use Windows system sounds from:
C:\Windows\Media\

Example commands to copy a Windows sound:
copy "C:\Windows\Media\Windows Notify System Generic.wav" alert.wav
(Then convert to MP3 using online converter or ffmpeg)
